# Аналитика продаж кофейни (SQL + Power BI)

## 📌 Описание
Проект по анализу продаж кофейни с использованием SQL и Power BI.

## 🎯 Цель
Построить витрину данных и визуализировать ключевые метрики:
- выручка
- количество заказов
- средний чек
- анализ продуктов

## 🛠 Стек
- PostgreSQL
- SQL
- Power BI

## 📊 Что сделано
- создана таблица order_metrics
- рассчитаны ключевые метрики
- построен дашборд

## 📈 Выводы
1. Выручка:
Общая выручка за период составила ~12.9 тыс. По дням наблюдаются колебания без явного тренда, но есть отдельные пики, что может быть связано с повышенным спросом в отдельные дни.

2. Заказы:
Количество заказов в целом стабильное. Рост выручки, если он есть, происходит не за счет увеличения среднего чека, а за счет количества заказов.

3. Продукты:
Основную выручку генерируют Mocha, Flat White и Latte. Это ключевые продукты, на которых держится большая часть продаж.

4. Клиенты:
Обнаружено 99 повторных клиентов, что говорит о наличии лояльной аудитории и повторных покупок.
